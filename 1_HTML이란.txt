컴퓨터는 비트단위로 처리한다.
파일 확장자 : 파일명 뒤에 . 이후에 들어가는 확장 이름, 해당 파일들을 사용하는 방법/정의

브라우저 : HTML문서의 프로그래밍 언어를 번역해서 렌더링해주는 소프트웨어/프로그램
    - 종류 : 크롬, 사파리, 엣지(구 익스플로러) 등등

HTML이란?
    1. H(Hyper) T(Text) M(Markup) L(Language) : 하이퍼텍스트 마크업 언어
        - 하이퍼(이동이가능한) 텍스트(문자) : 서로 다른문서를 연결 해주는 링크를 의미
            https://www.naver.com ( HTML 문서 )
            https://nid.naver.com/nidlogin.login (HTML 문서 )
        - 마크업 : 구조(HTML언어를 만든사람이구조화한)를 표시한다. 규칙/약속
        - 언어 : 프로그래밍 언어, 컴퓨터(0 or 1만 이해)가 이해(번역) 할 수 있는 언어
            HTML 언어를 기계어(0 OR 1)로 번역해서 컴퓨터가 이해한다.
            개발자(HTML) <-- 브라우저(컴파일러, 렌더링) --> 컴퓨터(기계어)
    2. HTML 파일 생성
        1. 폴더 선택( HTML 파일을 생성할 폴더 ), 오른쪽 클릭
        2. 새파일
        3.
        - 폴더 구조
            tj2024b_front ( 최상위 폴더 - 모든코드 파일들을 저장하는 폴더 )

                1_HTML이란.txt
    3. 주석이란?
        - 컴퓨터가 실행하지 않는 구역
        - 사용처 : 
            1. 협업 ( 코드 설명 공유 ) / 유지보수
            2. 공부 ( 코드 설명 기록 )

        - HTML : <!-- 주석 내용 -->
    4. HTML 실행
        - HTML문서의 코드(내용)이 변경되면 저장하고 실행하기
        방법1(로컬) : 폴더내 직접 html 파일을 실행한다.
        방법2(서버) : 마켓플레이스에서 라이브서버 설치해서 사용하기
            시작할 html파일 문서에서 오른쪽 클릭 -> Open with Live Server 클릭
